# blizbase
a Pocketbase based WoW Guild aggregator.

for it to be useful create a ".env" file beside blizbase and supply the following Environment Variables.

CLIENT_ID=
CLIENT_SECRET=
GUILD_SLUG=
REALM_SLUG=
PB_SUPERUSER_EMAIL=
PB_SUPERUSER_PASSWORD=
SMTP_USERNAME=
SMTP_PASSWORD=
SMTP_HOST=
SMTP_PORT=

or supply them to the docker container jrsmile/blizbase:latest
